# nyxveil-site

這是一個簡單的 Cloudflare Pages 靜態網站專案骨架。

## 結構

```text
nyxveil-site/
├─ index.html
├─ css/
│  └─ style.css
├─ js/
│  └─ main.js
└─ images/
```

## 用途

- `index.html`：網站入口
- `css/style.css`：樣式
- `js/main.js`：互動邏輯
- `images/`：圖片素材

## 說明

`README.md` 只會出現在 GitHub repo 裡，不會直接顯示在網站頁面上。

## commit 工具

可以執行 `tools/commit.py` 來互動式建立 git commit。

```bash
python tools/commit.py
```
