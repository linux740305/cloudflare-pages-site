# Cloudflare Pages Starter

這個資料夾已經是可直接部署到 Cloudflare Pages 的最小靜態網站。

## 目前結構

- `index.html`：首頁

## 部署方式

1. 把這個資料夾放進 GitHub repository。
2. 到 Cloudflare Pages 建立新專案。
3. 連接該 repository。
4. 如果是目前這種純靜態網站：
   - `Build command` 留空
   - `Build output directory` 留空，或依 Cloudflare 畫面使用根目錄
5. 部署完成後即可取得網站網址。

## 本機修改

直接編輯 `index.html` 即可。
